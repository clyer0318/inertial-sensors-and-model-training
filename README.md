# inertial-sensors-and-model-training
Inertial sensors measure acceleration, rotation, and orientation using devices like accelerometers and gyroscopes. Model training uses this sensor data to teach machine learning systems to recognize motion patterns and make predictions.  

慣性感測器利用加速度計與陀螺儀來測量加速度、旋轉與方向。
再透過這些感測資料去訓練模型，讓機器學習系統辨識動作模式並進行預測。
